**Badge** — compact uppercase pill for rarity tags, status, area labels and counts.

```jsx
<Badge rarity="legendary">Legendary</Badge>
<Badge tone="gold">Quest</Badge>
<Badge tone="ok" outline>Online</Badge>
<Badge tone="danger">Elite</Badge>
```

Set `rarity` to tint to a loot tier, or `tone` (`gold · neutral · danger · ok`) for status. `outline` gives a hairline style.
